## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/aMIrmxc/amirmxc-p1.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser at `http://localhost:5173` to see the magic!
